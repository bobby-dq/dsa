# Algorithms

## Two Pointers

### When to use?

1. **Linear data structure**: The input data can be traversed in a linear way (array, linked list, string)
2. **Process pairs**: Processing two data elements in two distinct positions simultaneously
3. **Dynamic pointer movement**: Both pointers are moving independently of each other according to certain conditions or criteria, moreover, both pointers might move along the same or two different data structures

## Sliding Window

### When to use?

1. **Contiguous data**: The input is stored in a contiguous manner, such as an array or string
2. **Processing subsets of elements**: The problem requires repeated computations on a contiguous subset of data elements, such that the window moves across the input array from one end to the other. May be fixed or variable
3. **Efficient computation time complexity**: The computations perfomed every time the window moves take constant or very small time
